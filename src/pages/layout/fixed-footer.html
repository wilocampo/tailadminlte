Fixed Footer Layout

This page demonstrates the fixed footer layout variant. The footer stays pinned at the bottom of the viewport while the main content scrolls above it. This is enabled by adding the fixed-footer class to the <body> tag.

Usage: <body class="fixed-footer layout-fixed sidebar-expand-lg sidebar-mini sidebar-open">

Try It

Toggle the checkbox and scroll down the page. When enabled, the footer remains visible at the bottom of the viewport. When disabled, the footer only appears at the end of the content.

How It Works

The fixed-footer body class activates CSS rules that make the .app-footer element position: sticky with bottom: 0. This keeps the footer anchored to the bottom of the viewport regardless of scroll position.

CSS Rules (from app-footer.css):
.fixed-footer .app-footer {
  position: sticky;
  bottom: 0;
  z-index: var(--z-fixed);
}
