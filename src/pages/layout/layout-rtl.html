RTL (Right-to-Left) Layout

This page demonstrates the RTL layout for languages like Arabic, Hebrew, Farsi, and Urdu. The sidebar appears on the right, text flows right-to-left, and all UI elements are mirrored.

Usage: Add dir="rtl" and lang="ar" to the <html> tag.

How to Enable RTL

To enable RTL layout, two changes are needed:

  1. Set the dir and lang attributes on the <html> element: 
    <html lang="ar" dir="rtl">
  2. Include the RTL CSS overrides that mirror the grid layout, sidebar position, and icon directions: 
    [dir="rtl"] .app-wrapper {
  grid-template-areas:
    "header sidebar"
    "main sidebar"
    "footer sidebar";
  grid-template-columns: 1fr auto;
}

Tailwind CSS utilities like text-right, ml-*, and pl-* are physical properties. For RTL support, prefer logical properties: ms-* (margin-start), me-* (margin-end), ps-* (padding-start), pe-* (padding-end).

English Content

This is a sample card with English content in the RTL layout. Notice how the overall page structure is mirrored — sidebar on the right, content flowing right-to-left — but the English text within this card still reads left-to-right naturally.

Mixed-language content works correctly because browsers handle bidirectional text natively.
