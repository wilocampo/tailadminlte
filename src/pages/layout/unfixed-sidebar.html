Default (Unfixed) Sidebar Layout

This page demonstrates the default sidebar layout. The sidebar scrolls along with the page content — it is not fixed to the viewport. This is the behavior when the layout-fixed class is absent from the body tag.

Usage: <body class="sidebar-expand-lg sidebar-mini sidebar-open">

Try It

Toggle to compare: when unchecked (default), the sidebar scrolls with the page. When checked, the sidebar becomes fixed.

How It Works

By default (without the layout-fixed class), the sidebar is part of the normal page flow. As you scroll down, the sidebar scrolls away with the content.

This is ideal for shorter sidebar menus or when you want to maximize vertical space for the sidebar content without needing an internal scrollbar.

Comparison:
Feature Default (Unfixed) Fixed
Body class — (no class needed) layout-fixed
Sidebar scrolls with page Yes No
Sidebar has own scrollbar No Yes
Best for Short menus Long menus, dashboards
